This is a WhatsApp group chat analysis using python, numpy, matplotlib, scikit learn, seaborn, wordcloud, NLP.
In this the raw text data was extracted from one of my class group.
Then the data was cleaned and preprocessed.
Then using data analysis libraries the chat was analised and insightfull data was extracted such as 
1. common topics
2. most active member
3. most active hours
4. most active day
5. least active member and so on.
